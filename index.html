perm_identity About me

Nimble table tennis player, skillful coffee-maker and web developer passionate about both front-end and back-end technologies. In a team environment, my peers often perceive me as a helpful and accommodating person. Having the ability to create visually appealing web apps from scratch that are useful to others simply makes me happy. Lifelong learner, I keep myself up to date with the latest development trends, with the goal of excelling as a developer.

Originally from small island of Mauritius, I came to Australia to study a bachelor of commerce in 2010 and stayed here since then. I am now pursuing my dream of being a Full-Stack Web Developer who loves creating beautiful and responsive web applications. I have a finance background via my commerce degree so I am very comfortable working with large data and big figures. Being business oriented by nature, I approach any task with the company's bottom line in mind. Finally, I am a very curious person and this enables me to consistently strive to learn the best and most effective way to do my work.

On the technology side, I have experience working with both front-end languages such as HTML5, CSS3, JS, jQuery and React.js and back-end languages like Ruby and Rails, Node.js, Express.js. I am also well versed with managing databases with SQL using Postgres. As such, I see myself as a full-stack developer as I enjoy working on both the front-end and back-end side of things.

view_module Latest Projects

Tic-Tac-Toe

This my first project at General Assembly. It’s an easy to use 2 player game that can be run straight from the browser. The focus of this project was on responsiveness and simple design.

Decision Maker

A full-stack app about helping people make up their mind from different options they have by enabling the user to sign-up for an account and login. They can then post a question with up to 4 different options that other registered users can vote on. Guests are only allowed to view the threads but not take any actions.

Use My Kitchen

An Airbnb like web app where users can rent out their unused kitchens and dining spaces. Other users can rent it for the night if they are organising a dinner party and don’t have the space for it. Built over one week in a team of 5. My main role in this project was creating the backend API end-point, database design and google maps integration.

WDI-conf

A single page app (SPA) consisting of a brochure site and a ticket-booking app for a hypothetical tech conference. The main aim of this project was to create a responsive app. All the views were written in javascript(jQuery) and then exported as separate modules.

Como Parade Café

Noticed my local café’s website was not looking so great and the information on it was out of date so I approached them to redesign it for them for free. I’ve used bootstrap for the basic layout and navigation bar and I’ve added some sprinkles of JS to it to keep it interesting.
